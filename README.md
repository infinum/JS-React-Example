# React Example project

## Project setup

```bash
# Install dependencies
npm ci

# Start the dev server
npm run dev
```

Use one of the user credentials to log in into the application

```
# user1
    email: 'user1@example.com'
    password: 'pasword12345'

# user2
    email: 'user2@example.com'
    password: 'pasword12345'

# user3

```
